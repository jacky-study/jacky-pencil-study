# jacky-pencil-study

> 学习 Pencil 工具的使用与技巧

## 项目简介

Pencil 是一个开源的 GUI 原型设计工具，本项目通过实际操作学习其使用方法。

## 目录结构

```
jacky-pencil-study/
├── official-demo.pen    # 官方示例
├── pencil-new.pen       # 新建模板
└── pencil-welcome.pen   # 欢迎页模板
```

## 学习内容

- Pencil 工具基本操作
- 原型设计文件结构
- `.pen` 文件格式分析
